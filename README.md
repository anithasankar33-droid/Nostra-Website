# 👕 Nostra – Considered Clothing

A modern, responsive multi-page fashion website for **Nostra**, a fictional slow-fashion clothing brand. The project is built using **HTML5, CSS3, and Vanilla JavaScript** with a clean UI and fully responsive design.

🌐 **Live Demo:**  
https://anithasankar33-droid.github.io/Nostra-Website/

---

## ✨ Features

### 🏠 Home Page
- Modern hero section
- Featured products
- Brand highlights
- Responsive navigation

### 🛍️ Collections Page
- Dynamic product catalogue
- Live product search
- Category filters
- Real-time results counter
- Responsive product grid

### 📩 Contact Page
- Client-side form validation
- Error messages
- Success confirmation
- Clean contact layout

### 📱 Responsive Design
- Mobile-friendly layout
- Hamburger navigation
- Tablet & desktop support
- CSS Grid & Flexbox

---
